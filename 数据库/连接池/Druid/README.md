# Druid

Druid 是由阿里巴巴数据库事业部出品的开源项目。它除了是一个高性能数据库连接池之外，更是一个自带监控的数据库连接池。虽然 HikariCP 已经很优秀，但是对于国内用户来说，可能对于 Druid 更为熟悉。所以，对于如何在 Spring Boot 中使用 Druid 是后端开发人员必须要掌握的基本技能。
