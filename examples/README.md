# spring-snippets

Spring & Java code snippets.

Spring Snippets 是 [Backend-Boilerplates](https://github.com/wx-chevalier/Backend-Boilerplates) 的一部分。

## Nav

- 如果你是想寻找项目模板，那么建议查阅 `spring-boilerplate-*` 系列项目。如果你是想寻找能够直接用于生产环境的项目模板，那么建议前往 [UDLA](https://github.com/BE-Kits/UDLA)。
